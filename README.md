## Axis SMP Mod

A Fabric mod made specifically for the Axis SMP.

Features:
- Arc abilities with cooldowns
- Rare artifacts obtained through exploration
- Custom structures, minibosses, and recall mechanics
- Basically vanilla+


### FAQ:
**Does this add permanent abilities?**  
No. All arcs are temporary and have cooldowns.

**Can I use this on my own server?**  
You absolutely can, but support and updates are not guaranteed.

**Does this replace vanilla progression?**  
No. Vanilla progression stays intact. Arcs are optional and exploration-based.

**What license does it use?**
GPLv3.
